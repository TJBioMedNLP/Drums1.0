What is Drums?

The full name of DRUMS is "Disease Related Unique Mutation Search Engine". Drums is a 'Bio-Google' in Mutation, Gene, Phenotype, LSDB Field. It was developed by 'BMInfo Group' from Tongji University and SCBIT in 2009.

What are the aims of Drums?

The overall purpose of Drum is collecting mutation records from the world's LSDBs and providing a proper search engine for checking whether a mutation record has been published or not. This aim is pursued by the following steps:

  • Use HGNC gene as standard of gene symbol.
  • Collect LSDB URLs as many as possible.
  • Different python scripts were written to extract mutation records automatically from LSDB.
  • Check the mutation information mannually, recorded in HGVS standard format.
  • Store all the data in Mysql database, create appropriate search method.
  • Establish Web-based interface to serve.
