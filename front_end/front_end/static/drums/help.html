FAQs:

1. Which browsers the system support?
We have tested the system on Chrome 10.x, Firefox 3.x, IE 8.0. For speed reason,
the Chrome or Firefox was suggested.

2. Will DRUMS have new version in the future?
Definitely. We will regularly update our database and keep the consistency with LSDBs.
In addition, we are still contacting curators of LSDBs to get the copyright permission.

3. What are the aims of Drums?
The overall purpose of Drum is collecting mutation records from the world's LSDBs
and providing a proper search engine for checking whether a mutation record has been
published or not. This aim is pursued by the following steps:
  1. Use HGNC gene as standard of gene symbol.
  2. Collect LSDB URLs as many as possible.
  3. Different python scripts were written to extract mutation records automatically from LSDB.
  4. Check the mutation information mannually, recorded in HGVS standard format.
  5. Store all the data in Mysql database, create appropriate search method.
  6. Establish Web-based interface to serve.
